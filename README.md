# The Manhole Matrix

A web application to help prevent accidents caused by open manholes, allowing users to:

- Register and log in
- Report accidents or issues
- View real-time updates

Built with:

- React.js (frontend)
- Node.js + Express.js (backend)
- MongoDB (database)
